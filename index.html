ShareIt.js

With ShareIt, you can easily create share buttons without copy/paste hastle. You can create share buttons with data attributes, or with JQuery.

This package uses FontAwesome for the icons and Bootstrap for the button classes, however, you're free to edit these settings in the settings.json file and rebuilding the package.

Share buttons with data attributes

These buttons are created using data-attributes. This way, you only need to import the shareit.js script, and don't need to write your own javascript.

Share buttons with JQuery

These buttons are created using jQuery. You can use the same options as you can with data attributes, and you can specify which elements you want to add share elements to. This is great for adding the same share elements to multiple places.
